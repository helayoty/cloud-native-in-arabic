# Cloud Native in Arabic

A hands-on learning repository for Cloud Native technologies, designed for Arabic speakers.

## What's Inside

This repository contains practical demos and tutorials covering container fundamentals, Docker, and cloud-native application development.

| Folder | Description |
|--------|-------------|
| [containers/](./containers/) | Learn how containers work under the hood - Linux namespaces, cgroups, and building containers from scratch |
| [docker/](./docker/) | Docker networking examples and scripts |
| [mininote-demo/](./mininote-demo/) | A complete Spring Boot + MongoDB application with Docker and docker-compose |

---

## Quick Start

### Clone the Repository

```bash
git clone git@github.com:helayoty/cloud-native-in-arabic.git
cd cloud-native-in-arabic
```
